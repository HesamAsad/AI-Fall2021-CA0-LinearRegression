Now df is our cleansed data and it's preprocessed. It doesn't have any NaN values anymore.

pros and cons of replacing NaN values with mean value:

  • pros:
    1. Mean value is the first and somehow best estimator for missing values
    2. It's Simple! Calculating the mean is really easy and has a low cost of calculation
  • cons:
    1. we can't use it for categorical data.
    2. Using mean for some numerical data would be meaningless. for example number of fingers of an animal couldn't be fractional.
    3. although it's easy to calculate, it may not be a good estimator in case of existing several outliers in the data.

Q10: Estimating function¶

Using Normal Equation:

In [ ]:
X0 = [1] * len(df_final)
X1 = list(df_final['FUELCONSUMPTION_COMB'])
X = np.array([X0, X1])
X = X.T
theta = np.linalg.inv(X.T @ X)
theta = theta @ X.T
theta = theta @ np.array(df_final['CO2EMISSIONS'])
In [ ]:
model = theta[0] + theta[1] * df_final['FUELCONSUMPTION_COMB']
